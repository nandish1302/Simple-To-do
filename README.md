# Simple To-Do App

A basic To-Do application built using **HTML, CSS, and JavaScript**.

## Features
- Add new tasks
- Delete tasks
- Stores data in localStorage
- Simple and clean UI

## How to Run
Just open `index.html` in any browser  
(or use Live Server in VS Code).

## Folder Structure
- index.html  
- style.css  
- script.js  

## Future Improvements
- Edit tasks
